# pages/

GitHub Pages root. Auto-deployed from `main` of this repo.

```
pages/
├── index.html                 # lists all assessments (newest first)
└── assessments/
    └── <batch>.html           # one per assessment (the Will-feedback page)
```

The assessment page is generated by `src/publish.py`. Will opens it,
sees each mark, and clicks per-mark agree / disagree / free text. His
responses PUT to the paper's KVdb bucket.

The index page is a hand-written / Jimothy-maintained "dashboard"
listing all assessments with status badges (assessed / feedback received
/ corrections applied).
